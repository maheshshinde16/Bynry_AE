# Sample Test Execution Report

> Status: template only. The repository has not been executed against a real application because approved test credentials and environment access were not supplied.

## Execution metadata

| Field | Value |
| --- | --- |
| Environment | QA/Staging — supplied by CI |
| Browser/device | Chromium desktop or mobile web — supplied by CI |
| Build ID | Local or CI build ID |
| Started | Generated by CI |
| Result | Generated by CI |

## Expected artifacts

- `artifacts/junit.xml`
- Screenshot, trace, video, browser-console output, and failed API response for a failure

## Example command

```bash
pytest -m integration --browser chromium --device mobile-web
```

Actual pass/fail results should be attached only after execution with authorized QA/staging credentials.
